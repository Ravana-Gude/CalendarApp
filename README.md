A modern, responsive calendar web application built with React and Tailwind CSS.
It helps users view, add, and manage events in different views (monthly, weekly, yearly) and also track upcoming and completed events.


Features
================
Multiple Views – Monthly, Weekly, and Yearly calendar layouts.
Add Events – Simple modal to add new events instantly.
Event Categories – Separate pages for Upcoming Events and Completed Events.
Conflict Handling – See multiple events colliding on the same date.
Clean UI – Built with Tailwind CSS for a professional, minimal design.
Dynamic Data – Events loaded from events.json.
Fast & Responsive – Works across desktop and mobile screens.

Tech Stack
=============
Frontend: React
Styling: Tailwind CSS
Date Handling: Day.js
Version Control: Git + GitHub
